# 培训考试系统

## 项目介绍

培训考试系统是一款 Java + Vue 的前后端分离的企业培训考试系统。主要优点是开发、部署简单快捷、界面设计友好、代码结构清晰。支持 Web 端、微信小程序和移动端 App，能覆盖到 PC 和手机等设备。

## 功能模块

### 员工端功能

| 模块 | 介绍 |
| ---- | ---- |
| 登录 | 用户名、密码 |
| 注册 | 部门、用户名、密码 |
| 任务中心 | 管理员发布的部门任务，每个员工只能做一次 |
| 考试 | 题干支持文本、图片、数学公式、表格等 |
| 固定试卷 | 可重复练习、自行批改的试卷 |
| 时段试卷 | 在时间限制内，可重复练习、自行批改的试卷 |
| 每日一练 | 每日练习题目，巩固知识 |
| 考试记录 | 查看答卷记录和试卷信息 |
| 错题本 | 答错题目会自动进入错题本 |
| 个人信息 | 显示个人资料、修改头像 |
| 消息中心 | 接收管理员发送的消息 |

### 管理端功能

| 模块 | 介绍 |
| ---- | ---- |
| 登录 | 用户名、密码 |
| 主页 | 试卷总数、题目总数、用户活跃度、题目月数量 |
| 员工列表 | 显示系统所有员工，新增、修改、删除、禁用 |
| 管理员列表 | 显示系统所有管理员，新增、修改、删除、禁用 |
| 业务范围列表 | 业务范围查询、修改、删除 |
| 试卷列表 | 试卷查询、修改、删除 |
| 试卷创编 | 创建时段试卷、固定试卷、任务试卷 |
| 题目列表 | 题目查询、修改、删除 |
| 题目创建 | 支持单选题、多选题、判断题、填空题、简答题 |
| 每日一练 | 创建每日练习，查看员工完成情况，导出数据 |
| 任务列表 | 任务查询、修改、删除 |
| 消息列表 | 显示已发送的消息及已读信息 |
| 用户日志 | 显示所有用户日志 |

### 小程序 / 移动端功能

| 模块 | 介绍 |
| ---- | ---- |
| 登录 | 用户登录登出，微信自动绑定 |
| 注册 | 部门、用户名、密码 |
| 考试 | 题干支持文本、图片、数学公式、表格等 |
| 每日一练 | 每日练习题目 |
| 考试记录 | 查看答卷记录和试卷信息 |
| 错题本 | 答错题目自动进入错题本 |
| 个人信息 | 显示个人资料 |
| 消息中心 | 接收管理员发送的消息 |

## 技术栈

- 后端：Spring Boot 2.1.6 + MyBatis + MySQL
- 管理端：Vue 2 + Element UI
- 员工端：Vue 2 + Element UI
- 移动端：React Native (Expo)
- 小程序：微信原生

## 部署

```bash
# 后端构建
cd source/xzs
mvn clean package -DskipTests

# 管理端构建
cd source/vue/xzs-admin
npm install && npm run build

# 员工端构建
cd source/vue/xzs-student
npm install && npm run build

# 启动
java -jar source/xzs/target/xzs-3.9.0.jar --server.port=8000
```
